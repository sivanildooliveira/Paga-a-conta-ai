1.	Tela 1 – Início

	•	Fundo chamativo (pode ser escuro premium)
	•	Botões:
	•	INICIAR RODADA
	•	(Opcional) Configurações (som on/off, idioma PT/EN)

	2.	Tela 2 – Multi-Touch (principal)

	•	Fundo preto/escuro ou neon (visual “balada/bar”)
	•	Texto no topo: “Coloque seu dedo na tela”
	•	Ao tocar:
	•	surge um círculo animado no ponto do toque (pulse/glow)
	•	toca um som curto (pop/click)
	•	cada dedo recebe cor aleatória e/ou efeito (aura)
	•	Enquanto houver pelo menos 2 dedos:
	•	inicia automaticamente a contagem (ou botão “Valendo” opcional)
	•	Suspense:
	•	tocar música/efeito de tensão que cresce
	•	contagem invisível aleatória entre 5 e 8 segundos
	•	efeitos visuais leves (vibração/pulsação/zoom micro)

	3.	Tela 3 – Resultado

	•	Escolhe 1 dedo aleatório entre os que continuam tocando
	•	Animação:
	•	zoom no círculo vencedor
	•	confete / flash
	•	som de “tcharam”
	•	risada zoeira
	•	Texto gigante: “PAGA A CONTA AÍ”
	•	Animação contínua por 2–3s: carteira voando e/ou dinheiro voando
	•	Botões:
	•	NOVA RODADA
